❓ FAQ

Frequently asked questions

What is Malzz Hosting? +
Malzz Hosting is a premium management panel for Pterodactyl — the popular open-source game server management platform. It provides a beautiful dashboard to create, monitor, and manage servers and admin accounts.
Do I need an existing Pterodactyl installation? +
Yes, Malzz Hosting connects to your existing Pterodactyl installation via its Application and Client APIs. You'll need your Pterodactyl domain and API keys to set it up.
Can I deploy this on my own VPS? +
Absolutely. Malzz Hosting is a Node.js Express app that can be deployed on any VPS with Node.js 18+. The README includes full VPS deployment instructions.
Is Firebase required? +
Firebase is required for activity logs, notifications, analytics, and admin management storage. The setup guide in the README walks you through Firebase project creation and configuration.
How secure is the panel? +
Malzz Hosting uses JWT authentication, Helmet security headers, express-rate-limit to prevent brute-force attacks, input validation, and environment variable-based secrets. It's built production-ready.
